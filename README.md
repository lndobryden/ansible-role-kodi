Ansible Role for Kodi
=========

An Ansible role for installing Kodi and the tvheadend PVR add-on

Requirements
------------

none

Role Variables
--------------

None

Example Playbook
----------------

```
    - hosts: desktops
      roles:
         - lndobryden.kodi
```
